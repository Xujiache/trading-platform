## 操作指引

### Banner 管理
1. 进入「运营管理 → Banner管理」
2. 点击「新增Banner」，上传图片或填写 URL
3. 设置跳转类型：无跳转/外部链接/页面/活动
4. 设置展示位置：顶部(轮播)/中部(推荐位)
5. 设置排序权重，数值越小越靠前

### 模拟盘费用配置
1. 进入「运营管理 → 模拟盘费用」
2. 页面展示全部活跃品种，未自定义的标记为"同步真实盘"
3. 点击「配置」可设置独立的手续费、点差、隔夜费
4. 点击「恢复默认」恢复为真实盘设置
5. 点击「全部同步真实盘」覆盖所有已有配置

### 第三方接口管理
1. 进入「运营管理 → 接口管理」
2. 新增接口时填写名称/类型/地址/密钥/额外配置
3. 编辑已有接口时会先拉取完整详情（含密钥），密钥留空则不修改
4. 点击「检测」执行 HTTP 连通性检测
