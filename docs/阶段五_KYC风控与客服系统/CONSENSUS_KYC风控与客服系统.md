## 3. 技术实现方案

- 2FA：speakeasy (TOTP) + qrcode (二维码生成)
- 在线客服：HTTP轮询（3秒/5秒间隔）
- 风控预警：由floatingPnlUpdater定时检查保证金比例自动生成
- 强平引擎：保证金比例低于force_close_line时系统自动执行

## 4. 技术约束与集成方案

- KYC 图片上传使用 multer 存储到 uploads/kyc/
- 2FA 基于 speakeasy（TOTP协议）
- 在线客服采用轮询方式（3秒间隔），暂不使用 WebSocket
- 风控参数已接入 tradeEngine（最大杠杆、单品种最大持仓校验）

## 5. 任务边界限制

- 自动强平巡检已在本阶段实现（集成在floatingPnlUpdater中）
- 活动相关功能在阶段六
