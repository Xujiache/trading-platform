## 环境依赖

| 配置项 | 说明 | 当前状态 |
|--------|------|----------|
| MySQL | 数据库地址/端口/用户名/密码/库名 | 通过 .env 配置，默认 localhost:3306 |
| Redis | Redis 地址/端口/密码 | 通过 .env 配置，默认 localhost:6379 |
| JWT 密钥 | Token 签名密钥 | .env 中已有开发环境默认值 |
| SMTP 配置 | 邮箱服务器信息 | 可通过后台邮箱配置页面设置 |
| 文件存储 | 上传文件存储位置 | 本地 uploads/ 目录 |

## 操作指引

### 后端启动
```bash
cd 服务端
npm install
# 初始化数据库（首次运行）
npm run db:init
# 启动开发服务器
npm run dev
```

### 默认管理员账号
- 用户名: `admin`
- 密码: `admin123`

### 管理后台启动
```bash
cd 管理后台
pnpm install
pnpm run dev
```

### 开发环境验证码
开发模式下如果 SMTP 未配置，验证码会打印在后端控制台，可直接使用。
