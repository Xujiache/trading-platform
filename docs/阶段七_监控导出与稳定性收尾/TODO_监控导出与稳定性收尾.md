## 操作指引

### 系统监控
- 管理后台 → 系统管理 → 系统监控
- 支持自动刷新（10秒间隔）和手动刷新
- 显示DB/Redis/连接池/行情源/系统资源信息

### 数据导出
- 管理后台 → 系统管理 → 数据导出
- 支持6种导出类型：用户列表、订单列表、资金流水、入金记录、出金记录、交易流水
- 支持按日期和状态筛选

### 批量操作
- 管理后台 → 系统管理 → 批量操作
- 三个Tab：批量确认入金、批量审核出金、批量冻结/解冻用户
- 输入ID列表（逗号/换行分隔），执行后显示成功/失败明细

### 审计日志
- 管理后台 → 系统管理 → 审计日志
- 支持按模块/操作/操作人/状态/日期筛选
- 详情弹窗显示完整操作信息
